## 기능 명세
### 입력
- [ ] 사용자 입력 기능 구현
    - [ ] 다리 길이 입력
      - 예외 상황
          - [X] 3 이상 20 이하의 숫자가 아닐 때
    - [ ] 이동할 칸을 입력
      - 예외 상황
          - [X] U(위 칸)와 D(아래 칸) 중 하나의 문자가 아닐 때
  - [ ] 게임 재시작 여부 입력
      - 예외 상황
          - [X] R(재시작)과 Q(종료) 중 하나의 문자를 입력이 아닐 때
- [X] 공통 예외
  - [X] blank 입력이 들어올 수 없다

### 서비스 로직
- [ ] 다리를 생성 기능
- [ ] 플레이어가 입력된 값으로 다리 이동 하는 기능
- [ ] 이동한 결과를 구하는 기능
